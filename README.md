# TCG Lightning

![TCG Lightning](static/logo_full.png)

A modern, lightning-fast deck building application for Magic: The Gathering.

**[Visit the Website for more information](https://dimfred.github.io/tcg-lightning/)**

## What is TCG Lightning?

TCG Lightning is a powerful deck building tool designed for Magic: The Gathering players who want a fast, intuitive, and feature-rich experience. Built with modern web technologies, it provides an efficient way to search cards, build decks, and manage your collection.

## Key Features

- **Lightning-Fast Search**: Quickly find any card from Magic's extensive library
- **Modern Interface**: Clean, responsive design that works seamlessly across devices
- **Intuitive Deck Building**: Build and organize your decks with ease
- **Custom Proxy Creation**: Create your own proxies with custom artwork directly in the app - perfect for playtesting or personalizing your decks

## Installation

### Arch Linux Users

Install via the AUR:

```bash
paru -Sy tcg-lightning-bin
```

### Other Platforms

Download the latest release for your platform from the [Releases](https://github.com/dimfred/tcg-lightning-public/releases) page.
