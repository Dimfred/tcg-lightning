# 0.1.25

## Features

- friends & groups - share your collection with friends and create groups for collaborative deckbuilding
- group collections - view and manage shared collections within groups
- collection management - create, rename, delete, and filter collections
- collection sync - real-time synchronization of collections across devices
- drag & drop tabs and collections - reorder tabs and collections with drag and drop
- maybeboard toggle in friends/groups search view

## Bug Fixes

- infinite scroll now works properly

- fixed foil card display issues
- fixed autocomplete not triggering on LLM syntax search
- fixed auth screen flickering
- fixed drag & drop on Windows
- fixed rendering issues on Linux
- fixed manabox import deduplication bug
